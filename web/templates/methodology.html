Methodology & Transparency

🔍 Overview

The AI Forex Signal Hub uses a fully automated, no-cost machine-learning pipeline to generate daily buy/sell/hold signals for three major forex pairs. All components run within their respective free-tier limits and are designed to be transparent and reproducible.

Signals are generated once per day after the New York session closes (approx. 21:00 UTC) and are published to a central Google Sheet, which this site reads via the Sheets API.

📡 Data Sources

  • Historical OHLCV prices: Twelve Data API (free tier – 800 API calls/day) and Yahoo Finance via the yfinance Python library. 10+ years of daily candles are stored in the Google Sheet Training_Data tab.
  • News sentiment: NewsAPI.org (free tier – 100 requests/day). Article titles and descriptions are scraped for the pairs' base currencies and scored using a simple VADER-based polarity model. Results are stored in the News_Sentiment tab.
  • Economic indicators: CPI delta, PMI, and interest-rate differentials are collected monthly via Zapier webhooks from freely available government statistical agencies (BLS, ONS, Eurostat, BoJ).

🤖 Machine Learning Model

The model is a LightGBM gradient-boosted tree classifier trained to predict the next-day direction (BUY / SELL / HOLD) for each pair. Features include:

  • RSI-14, MACD (12/26/9), EMA-20, EMA-50, Bollinger Band width
  • Daily return, 5-day / 20-day momentum
  • Aggregated news sentiment score (rolling 3-day)
  • CPI delta, PMI, yield spread (base vs. quote currency)

The model is retrained weekly using a Google Colab notebook (free GPU runtime). The notebook reads the Training_Data sheet, trains LightGBM with 5-fold time-series cross-validation, and writes the latest predictions to the Predictions sheet.

# Simplified training snippet (Google Colab) import lightgbm as lgb from sklearn.model_selection import TimeSeriesSplit model = lgb.LGBMClassifier( n_estimators=400, learning_rate=0.05, num_leaves=31, class_weight='balanced' ) tscv = TimeSeriesSplit(n_splits=5) # ... fit, evaluate, write predictions to Google Sheets

Alternatively, Obviously AI or Akkio (both free-tier) can be connected directly to the Google Sheet to train and output predictions without writing any code.

⚙️ Automation Pipeline

The end-to-end pipeline is orchestrated with Zapier (free tier – 100 tasks/month):

  • Step 1 – Data collection: Daily Zapier schedule triggers a webhook that calls Yahoo Finance / Twelve Data and appends new rows to Training_Data.
  • Step 2 – News scraping: Zapier RSS / NewsAPI zap appends the latest headlines with sentiment scores to News_Sentiment.
  • Step 3 – Model inference: The Colab notebook (scheduled weekly via Google Cloud Scheduler free tier) retrains and publishes predictions to Predictions.
  • Step 4 – Alert dispatch: A Zapier "New Row" trigger on Predictions fires a Twilio SendGrid email (free – 100 emails/day) to all subscribers.

⚠️ Limitations & Black-Box Notice

LightGBM is an inherently black-box model: while feature importances can be inspected, it does not provide human-readable reasoning for individual predictions. The confidence score is derived from the model's predicted class probability, not a rigorous uncertainty estimate.

Forex markets are influenced by events that cannot be captured in historical price data or news sentiment alone (e.g. geopolitical shocks, central bank surprises). The model may underperform during high-volatility or news-driven regimes.
